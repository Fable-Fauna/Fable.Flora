# Flora.CssParser
A to spec css parser for .NET.

# Flora.CssProvider
A type provider for consumption in both .NET and Fable runtimes to provide class names. And themeing support via auto injecting css varibles.

# Supported Css 
* Tailwind (currently working)
* Bootstrap 
* Pure 
* Bulma
* Milligram
* Semantic-UI
* Foundation
* Material UI
* UIKit
